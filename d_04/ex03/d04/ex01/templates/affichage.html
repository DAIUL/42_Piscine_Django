Processus d’affichage d’une page statique

  1. Le navigateur envoie une requête HTTP au serveur Django
  2. Django reçoit la requête et cherche l’URL dans urls.py pour savoir quelle vue appeler
  3. La vue associée est exécutée. Pour une page statique, elle se contente de rendre un template
  4. Le template est transformé en HTML complet et renvoyé dans une réponse HTTP
  5. Le navigateur reçoit la réponse et affiche la page

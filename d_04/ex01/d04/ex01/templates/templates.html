Moteur de template

Le moteur de templates de Django permet de séparer le code Python de la présentation HTML.
Il prend un fichier template et remplace les variables et instructions de contrôle par leur contenu réel pour générer une page HTML complète.

Blocs

Un bloc définit une zone du template pouvant être remplacée ou complétée par un template enfant lorsqu’on utilise l’héritage de template ({% verbatim %}{% block nom_du_bloc %}{% endblock %}{% endverbatim %}).

Boucles for...in

Elles permettent de parcourir une liste ou un dictionnaire passé au template et d’afficher dynamiquement son contenu ({% verbatim %}{% for item in liste %} ... {% endfor %}{% endverbatim %}).

Structures if

Elles permettent d’afficher du contenu conditionnellement selon la valeur d’une variable ({% verbatim %}{% if condition %} ... {% else %} ... {% endif %}){% endverbatim %}.

Affichage de variables

Les variables passées au template via le contexte sont affichées avec les doubles accolades ({% verbatim %}{{ variable }}{% endverbatim %}).
